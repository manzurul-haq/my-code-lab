My Code Lab as a beginner web developer

I started with HTML, which are the building blocks of web pages. I created simple web pages with headings, paragraphs, and images. Now I write about HTML language. HTML full form is HyperText Markup Language. It is used to create the structure and content


Here are some basic HTML tags:
  1. How we create table using HTML tags:

    <table> : This tag is used to create a table.
    <tr> : This tag is used to create a table row.
    <th> : This tag is used to create a table header cell.
    <td> : This tag is used to create a table data cell.

    Example:
    Name Age City
    Akash 25 Dhaka
    Sujo 18 Jashore

  2. How we create a link using HTML tags:

    <a> : This tag is used to create a hyperlink.
    href attribute: This attribute is used to specify the URL of the page the link goes to.

    Example: Click here to visit My Linkdin Account

  3. How we create a list using HTML tags:

    <ul> : This tag is used to create an unordered list.
    <ol> : This tag is used to create an ordered list.
    <li> : This tag is used to create a list item.

    Example:
    Unordered List:
    • Apple
    • Banana
    • Orange
    Ordered List:
    1. First Item
    2. Second Item
    3. Third Item
